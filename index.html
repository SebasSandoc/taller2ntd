La arquitectura de Software

La arquitectura de software se refiere a la estructura fundamental de un sistema de software, incluyendo sus componentes, las relaciones entre esos componentes, y los principios que guían su diseño y evolución. Es una descripción de cómo un sistema está organizado y cómo las partes del sistema interactúan entre sí.

A Distributed Software Architecture for IoT: Container Orchestration Impact and Evaluation

The necessity of a DSA in modern IoT solutions, emphasizing its ability to enhance scalability, reduce latency, and improve security through distributed processing. The research is structured around a two-phase methodology: the first phase involves simulating scenarios using synthetic data across public and private cloud environments, while the second phase focuses on deploying the system in a private cloud using real sensor data. Key findings from performance tests indicate that utilizing a container orchestrator significantly improves distributed processing capabilities, leading to lower latency, increased flexibility, enhanced security, and overall cost-effectiveness. The architecture is designed to be adaptable to various cloud environments and network topologies, making it suitable for diverse IoT applications, including smart homes, smart campuses, and health-related systems. The importance of reproducibility in research by making the architecture's source code publicly available. This contributes to the knowledge base in the field of IoT by providing a comprehensive analysis of architectural characteristics and benchmarking, ultimately proposing a versatile and modern DSA that aligns with current cloud computing trends. articulo

Authors: Gustavo M. Freirea, Herminio Paucar Curasma, Julio Cezar Estrella

Keywords: Distributed architecture, reactive manifesto, cloud computing, microservices, IoT

Publication Date: April 23-25, 2024

Published in: The 15th International Conference on Ambient Systems, Networks and Technologies(ANT)

Main Ideas:

  • Distributed Software Architecture (DSA): The paper proposes a DSA that adheres to reactive principles and is designed to be agnostic to various cloud computing models. This architecture is intended for deployment on edge computing devices and aims to support complex and scalable demands in IoT contexts, particularly for Smart Buildings (SB).
  • Performance Evaluation: The research includes performance tests comparing different deployment scenarios, specifically between public and private clouds. The results indicate that using a container orchestrator significantly enhances data reception capacity and system throughput, particularly through the implementation of replicas and load balancing techniques.
  • Impact of Cloud Environment: The findings suggest that hosting the system in a private cloud reduces latency compared to a hybrid configuration, which is crucial for maintaining system performance. The paper highlights that public cloud deployments can introduce latency overhead, negatively affecting performance metrics such as failure percentage.

Investigation and implementation of digital software architecture based on internet of things

Application systems are becoming increasingly complex, and software system architecture plays an increasingly important role in the software development process. In view of the problems of the traditional software architecture, such as slow data processing speed, poor stability of the architecture and low accuracy of data classification, this paper studied the digital software architecture based on the Internet of Things (IoT) technology, and conducted data pre-processing on the data collected by sensors, mainly including data reading, data cleaning, data transformation and data reduction. Through experiments, it could be found that the accuracy of digital software architecture using data preprocessing for data classification was more than 93.52%, and the average accuracy of 10 experiments was 94.55%. At the same time, the digital software architecture based on the IoT was tested, and it was found that both the execution time and the original memory were better than other digital software architecture. The overall performance of digital software architecture based on the IoT was better.articulo
